# About SALOME

## Overview
SALOME is a comprehensive platform for numerical simulation, offering:
- CAD modeling
- Mesh generation
- Multi-physics coupling
- Data visualization
- Workflow management

- EDF (Électricité de France)
- OCC (Open Cascade Technology)
